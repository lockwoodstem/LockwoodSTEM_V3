POE Unit 3

Autonomous Aerospace Systems: Controls, Drones & Fluid Power

Students explore how automated aerospace systems use logic, sensors, feedback, actuators, and fluid power to complete missions safely and reliably. Robolink drones and VEX ground-support systems become the platform for programming, testing, and refining open-loop and closed-loop control behavior.

Unit Outcome

Drone Mission + VEX Ground Support System

Final Evidence: Drone mission demonstration + VEX or fluid-power support system + flowcharts/code evidence + test data + final design review

Unit Purpose

What this unit prepares you to do

Unit 3 moves from control-system logic to physical mission systems. You will plan algorithms, test drone behavior, explore feedback and sensors, use VEX or fluid power for ground support, and defend an integrated autonomous mission with evidence.

Plan control logic

Create flowcharts, mission algorithms, and decision tables that show how an automated system should behave.

Use feedback

Compare open-loop and closed-loop control by testing drone missions, sensors, thresholds, and actuator responses.

Integrate systems

Combine a Robolink drone mission with a VEX, pneumatic, hydraulic, or physical ground-support system.

Unit Project

Drone Mission + VEX Ground Support System

Design, program, test, and demonstrate an autonomous drone mission supported by a VEX, fluid-power, or physical ground-support system. Your final design must use control logic, testing evidence, and mission performance data to justify decisions.

Possible missions

Aerial inspection route, landing-zone detection, payload delivery simulation, obstacle navigation, search-and-survey path, or approved custom aerospace mission task.

Support systems

VEX landing gate, payload station, rotating beacon, inspection platform, signal tower, hydraulic/pneumatic door, or mission checkpoint structure.

Required evidence

Problem statement, flowchart, system architecture diagram, code screenshots, build photos, test data, iteration notes, and final design review.
