Hands-on requirement

After passing the online test, complete a teacher-reviewed sketching task using class sketch paper or engineering notebook pages.

Student must demonstrate

  • Create a clear isometric sketch using vertical and 30° receding axes.
  • Create aligned orthographic views that communicate the same object.
  • Use visible, hidden, center, and construction lines appropriately.
  • Add labels, notes, and key features without cluttering the sketch.
  • Maintain proportion, neatness, and readable documentation.
  • Revise the sketch based on feedback when needed.

Badge unlock rule

  • Online test score of 80% or higher
  • Teacher-observed or teacher-reviewed hands-on completion
  • Readable technical sketching communication
  • Proper documentation and revision habits
