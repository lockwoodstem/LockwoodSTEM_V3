Hands-on requirement

After passing the online test, submit or show a teacher-reviewed Fusion model of a simple parametric part.

Student must demonstrate

  • Create and save the design in the correct project/location.
  • Start a sketch on an appropriate plane or face.
  • Use sketch constraints and dimensions to control geometry.
  • Create a valid closed profile and extrude it into a 3D feature.
  • Edit a sketch dimension or feature without rebuilding from scratch.
  • Use inspect tools and name/save/export the file correctly.

Badge unlock rule

  • Online test score of 80% or higher
  • Teacher-reviewed CAD model task
  • Clear evidence of parametric sketch/feature control
  • Proper file organization and export habits
