Hands-on requirement

After passing the online test, submit or show a teacher-reviewed documentation sample from an engineering task, prototype, test, or design decision.

Student must demonstrate

  • Create a dated engineering notebook entry with a clear title and purpose.
  • Document design constraints, key decisions, and reasoning.
  • Include useful sketches, photos, measurements, or test data as evidence.
  • Record results honestly, including failures or unexpected outcomes.
  • Track revisions so someone can understand what changed and why.
  • Use readable formatting, file naming, and organization.

Badge unlock rule

  • Online test score of 80% or higher
  • Teacher-reviewed hands-on documentation sample
  • Clear evidence-based engineering communication
  • Proper revision and file organization habits
